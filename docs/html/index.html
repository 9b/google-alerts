Python Google Alerts¶

The google-alerts Python module provides an abstract interface for the Google Alerts service. Google does not provide an official API for this service, so interactions are done through web scripting. Questions, comments or for support needs, please use the issues page on Github.

Features¶

  • Add new monitors (RSS or Mail)
  • Modify existing monitors
  • Delete monitors by ID or term
  • List all monitors with details
